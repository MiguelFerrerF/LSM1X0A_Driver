Detailed Description

High-level orchestration client for the LSM100A/LSM110A module.

This class wraps an internal instance of LSM1x0A_Controller and exposes an ultra-simplified interface for setting up and sending data via LoRaWAN or Sigfox, managing all the required initialization, recovery, and synchronization routines internally.
