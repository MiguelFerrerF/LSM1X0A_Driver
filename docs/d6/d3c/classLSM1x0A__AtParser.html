Obtains the operating mode detected during boot. This is useful to adapt the controller's behavior to differences between LoRaWAN and SigFox modes. The mode is determined by analyzing responses to specific AT commands during the initialization phase. If the module does not respond or cannot be identified, it returns LsmMode::MODE_UNKNOWN.

Returns
