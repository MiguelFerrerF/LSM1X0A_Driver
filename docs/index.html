LSM1x0A Driver Documentation

Introduction

Welcome to the official C++ Driver Documentation for the Seongji LSM1x0A Module. This library provides a robust, asynchronous, and event-driven interface to control the LSM1x0A radio module, utilizing its AT command architecture for both LoRaWAN and Sigfox networks over a serial link (UART).

Key Features

  • Dual-Stack Support: Seamlessly switch and operate between LoRaWAN and Sigfox modes.
  • Asynchronous Event Handling: Non-blocking handleEvent() loop to process incoming radio events.
  • Robust AT Parser: Intelligent queueing and recovery on UART timeouts or unresponsive hardware.
  • Modern C++ API: Strongly typed Enums, clear Structs, and encapsulated Object-Oriented design.

Navigation Hub

Explore the documentation using the tabs above or the tree-view on the side. The documentation is logically divided into these core modules:

  • **LSM1x0A_Client** : The high-level abstraction layer resolving network complexity for simple end-user applications.
  • LSM1x0A Hardware & Controller : The LSM1x0A_Controller class. Handles UART, AT commands, hardware resets, and the generic event loop.
  • LSM1x0A LoRaWAN API : The LSM1x0A_LoRaWAN API class. Handles OTAA/ABP Joins, Channels, Port routing, Confirmed/Unconfirmed payloads.
  • LSM1x0A Sigfox API : The LSM1x0A_Sigfox API class. Handles Sigfox Network Joins, Send Bits/Frames, OOB, and RF Tests.

Available Examples

The driver includes several practical examples to help you get started quickly:

  • **client_example.cpp**: Demonstrates the easiest way to use the library via the LSM1x0A_Client layer. Shows how to join a network, send data, and register a downlink callback.
  • **parser_example.cpp**: Demonstrates how to use the low-level LSM1x0A_AtParser directly to send raw AT commands and parse custom events.
  • **lorawan_example.cpp**: A comprehensive test suite that interacts with the LSM1x0A_Controller directly to test all LoRaWAN getters, setters, and operational state machines.
  • **sigfox_example.cpp**: A comprehensive test suite specifically designed for testing Sigfox specific operations like bits, frames, and out-of-band messages.

Hardware Overview

The LSM100A is a compact, low-power, dual-mode LPWAN module based on the STMicroelectronics STM32WL chipset. It natively supports LoRaWAN® Class A/C operations as well as Sigfox (RC1/RC2/RC3/RC4) protocol stacks.
