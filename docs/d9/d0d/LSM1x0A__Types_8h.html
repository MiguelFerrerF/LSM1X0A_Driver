◆ AtError

enum class AtError
strong

Represents the parsed AT command response codes and internal states of the LSM1x0A parser. Maps directly to errors thrown by lora_at.c and lora_command.c inside the firmware.

Enumerator
OK 

Success token (e.g. \r\nOK\r\n)

GENERIC_ERROR 

Generic LoRaWAN error (AT_ERROR)

PARAM_ERROR 

Parameter error, out of bounds or malformed (AT_PARAM_ERROR)

BUSY 

Module is busy. Important: Usually indicates a backoff/retry is needed (AT_BUSY_ERROR)

TEST_PARAM_OVERFLOW 

Parameter query overflow error (AT_TEST_PARAM_OVERFLOW)

NO_NET_JOINED 

Tried to send data without being joined to network. Important: Trigger Join (AT_NO_NETWORK_JOINED)

RX_ERROR 

Serial or radio reception error (AT_RX_ERROR)

NO_CLASS_B_ENABLE 

Attempted Class B operation without it being enabled (AT_NO_CLASS_B_ENABLED)

DUTY_CYCLE_RESTRICT 

Transmission restricted due to ETSI Duty Cycle limits (AT_DUTYCYCLE_RESTRICTED)

CRYPTO_ERROR 

Cryptographic MAC error (AT_CRYPTO_ERROR)

LIBRARY_ERROR 

Internal SigFox library error (AT_LIB_ERROR)

TX_TIMEOUT 

Transmission timeout, channel busy or CS (AT_TX_TIMEOUT)

RX_TIMEOUT 

Reception window timeout (AT_RX_TIMEOUT)

RECONF_ERROR 

Reconfiguration error (AT_RECONF_ERROR)

BOOT_ALERT 

Boot alert received spontaneously ("BOOTALERT"). Indicates module was restarted.

TIMEOUT 

The parser timed out waiting for the final \r\nOK\r\n token.

BUFFER_OVERFLOW 

Internal string buffer overflowed.

UNKNOWN 

Unknown AT string returned by the module.
