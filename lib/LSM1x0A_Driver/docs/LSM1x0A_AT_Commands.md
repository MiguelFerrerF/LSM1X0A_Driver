# LSM1x0A AT Commands Reference

This document compiles all available AT commands for the LSM1x0A module, both for its LoRaWAN and Sigfox firmware. The module receives these commands via its serial interface (TX/RX).

---

## Serial Interface (UART)
- **Baudrate Range**: 9600 (default) or 115200.
  - *Note:* To operate in Sigfox mode, the baudrate **must** be set to 9600.
- **Command Format**: All commands begin with `AT`. Some use the format `AT+<CMD>`, `AT$<CMD>` or `AT<CMD>`.
- **Common Syntax**:
  - `AT+<CMD>?` : Help for the command.
  - `AT+<CMD>` : Execute the command.
  - `AT+<CMD>=<value>` : Set the value.
  - `AT+<CMD>=?` : Get the current value.

| :--- | :--- |
| `\r\nAT_PARAM_ERROR\r\n` | Incorrect parameters, out of range or malformed. |

### Specific LoRaWAN Responses
| Response | Description |
| :--- | :--- |
| `\r\nAT_NO_NETWORK_JOINED\r\n` | Attempt to send data without having joined the network (JOIN). |
| `\r\nAT_NO_CLASS_B_ENABLE\r\n` | Attempt to operate in Class B without it being enabled. |
| `\r\nAT_DUTYCYCLE_RESTRICTED\r\n` | Restricted due to exceeding the ETSI Duty Cycle of the channel/band. |
| `\r\nAT_CRYPTO_ERROR\r\n` | General error in the LoRaWAN MAC cryptographic layer. |

### Specific Sigfox Responses
| Response | Description |
| :--- | :--- |
| `\r\nAT_LIB_ERROR\r\n` | Internal error in the Sigfox library. |
| `\r\nAT_TX_TIMEOUT\r\n` | Transmission timeout elapsed (Channel busy/CS). |
| `\r\nAT_RX_TIMEOUT\r\n` | Reception timeout elapsed. |
| `\r\nAT_RECONF_ERROR\r\n` | Error during reconfiguration. |

## Common AT Commands (System / Interface)

| Command | Help | Description / Usage |
| :--- | :--- | :--- |
| `ATZ` | `ATZ` | Reboots/Resets the processor (Trigger MCU reset). |
| `AT+RFS` / `AT$RFS` | `AT+RFS` or `AT$RFS` | Restores factory settings in EEPROM memory. |
| `AT+VL` | `AT+VL=<Level>` | Verbose/Debug Level. `[0:Off .. 3:High]`. |
| `AT+LTIME` | `AT+LTIME` | Gets local time in UTC format (LoRaWAN only). |
| `AT+BAUDRATE` | `AT+BAUDRATE` | Sets the UART baudrate. Allowed values: 9600, 115200. Requires reboot (LoRaWAN / general only). |
| `AT+BAT` | `AT+BAT` | Returns the battery level in millivolts (mV). |
| `AT+MODE` | `AT+MODE` | Selects the radio technology. `0:Sigfox`, `1:LoRa`. (Requires 9600 baudrate for Sigfox). |
| `AT+FW` | `AT+FW` | Activates firmware upgrade mode (Ota/Serial FW upgrade mode). |
| `AT+VER` | `AT+VER` | Returns the firmware version (FW version). |
| `AT$SSWVER` | `AT$SSWVER` | Returns the Stack software version (SW Version). |

## AT Commands: LoRaWAN

### 1. Keys, IDs and EUIs Management
| Command | Set Format | Description |
| :--- | :--- | :--- |
| `AT+APPEUI` | `AT+APPEUI=<XX:XX:XX:XX:XX:XX:XX:XX>` | (Or `+JOINEUI`) Gets or sets the Application EUI (Join EUI). |
| `AT+DEUI` | `AT+DEUI=<XX:XX:XX:XX:XX:XX:XX:XX>` | Gets or sets the Device EUI. |
| `AT+DADDR` | `AT+DADDR=<XXXXXXXX>` | Gets or sets the Device Address. |
| `AT+APPKEY` | `AT+APPKEY=<16 Bytes (Hex)>` | Gets or sets the Application Key. |
| `AT+NWKKEY` | `AT+NWKKEY=<16 Bytes (Hex)>` | Gets or sets the Network Key. |
| `AT+APPSKEY` | `AT+APPSKEY=<16 Bytes (Hex)>` | Gets or sets the Application Session Key. |
| `AT+NWKSKEY` | `AT+NWKSKEY=<16 Bytes (Hex)>` | Gets or sets the Network Session Key. |
| `AT+NWKID` | `AT+NWKID=<NwkID>` | Gets or sets the Network ID `[0..127]`. |

### 2. Join and Data Transmission (LoRaWAN)
| Command | Set Format | Description |
| :--- | :--- | :--- |
| `AT+JOIN` | `AT+JOIN=<Mode>` | Joins the network. Mode: `0 (ABP)`, `1 (OTAA)`. |
| `AT+LINKC` | `AT+LINKC` | Adds a *Link Check Request* (piggyback) to the next uplink. |
| `AT+SEND` | `AT+SEND=<Port>:<Ack>:<Payload>` | Sends a binary payload (hex) via application `Port [1..199]`. `Ack`: `0 (unconf)`, `1 (conf)`. |
| `AT+DEVNONCE` | `AT+DEVNONCE=0` | Gets or resets the DevNonce to 0. |

### 3. MAC Network Configuration (LoRaWAN)
| Command | Set Format | Description |
| :--- | :--- | :--- |
| `AT+BAND` | `AT+BAND=<BandID>` | Defines Active Region. `0:AS923, 1:AU915, ... 4:EU433, 5:EU868, ... 8:US915...` |
| `AT+CLASS` | `AT+CLASS=<Class>` | Defines Device Class. `A`, `B`, or `C`. |
| `AT+ADR` | `AT+ADR=<ADR>` | Adaptive Data Rate (ADR). `0:off`, `1:on`. |
| `AT+DR` | `AT+DR=<DataRate>` | Transmission Data Rate (Tx DataRate). `[0..7]`. |
| `AT+DCS` | `AT+DCS=<DutyCycle>` | ETSI DutyCycle (testing). `0:disable`, `1:enable`. |
| `AT+RX1DL` | `AT+RX1DL=<Delay>` | Rx Window 1 Delay in ms. |
| `AT+RX2DL` | `AT+RX2DL=<Delay>` | Rx Window 2 Delay in ms. |
| `AT+RX2DR` | `AT+RX2DR=<DataRate>` | Rx Window 2 Data Rate. `[0..7]`. |
| `AT+RX2FQ` | `AT+RX2FQ=<Freq>` | Rx Window 2 Frequency in Hz. |
| `AT+JN1DL`/`JN2DL`| `AT+JN1DL=<Delay>` | Join Accept Window 1 and 2 Delay in ms. |
| `AT+TXP` | `AT+TXP=<Power>` | Transmit Power `[0..15]` (range depending on region). |
| `AT+PGSLOT` | `AT+PGSLOT=<Period>` | Set/Get unicast ping slot Period `[0:1s .. 7:128s]`. |
| `AT+NWKTYPE` | `AT+NWKTYPE=<NetworkType>`| Config. Network Type. `0:Public`, `1:Private`. |
| `AT+CNFRETX` | `AT+CNFRETX=<NbTrans>` | Maximum *Confirmed* retransmissions `[1..15]`. |
| `AT+UNCNFRETX` | `AT+UNCNFRETX=<NbTrans>`| Maximum *Unconfirmed* retransmissions `[1..15]`. |
| `AT+CHMASK` | `AT+CHMASK=<Mask>` | Set/Get channel mask. |
| `AT+ABPFCNT` | `AT+ABPFCNT=<Fcnt>` | Get/Set the ABP frame counter. `[0..4294967295]`. |

### 4. Testing and Certification Mode (LoRa/RF)
*Note: These are generally used for RF tests, certifications and hopping.*
| Command | Description |
| :--- | :--- |
| `AT+TTONE` / `AT+TRSSI` | Starts continuous TX tone / RX RSSI test tone. |
| `AT+TCONF` | Set RF config `<Freq>:<Power>:<BW>:<SF/Dr>:<CR>:<Lna>...`. |
| `AT+TTX` / `AT+TRX` | Starts TX test transmissions (`AT+TTX=<Packets>`) / RX wait test (`AT+TRX=<Packets>`). |
| `AT+TTH` | Tx hopping test (`AT+TTH=<Fstart>,<Fstop>,<Fdelta>,<PacketNb>`). |
| `AT+TOFF` | Stops ongoing radio tests. |
| `AT+CERTIF` | Enters LoraWan Certification Mode `Mode [0:ABP, 1:OTAA]`. |
| `AT+MTX` / `AT+MRX` / `AT+CERTISEND` | Internal modulation tests and cert send. |
| `AT+PCONF` / `AT+PSEND` / `AT+PRECV` | Point-to-Point configuration / transmissions (P2P / pure Lora PHY). |

## AT Commands: Sigfox

### 1. Device and Identity Variables
| Command | Format | Description |
| :--- | :--- | :--- |
| `AT$ID` | `AT$ID` | Returns the device **Sigfox ID**. |
| `AT$PAC` | `AT$PAC` | Returns the initial **PAC** (Porting Authorization Code). |
| `ATS410` | `ATS410=<Mode>` | `0`: Private key, `1`: Public Key. |
| `ATS411` | `ATS411=<Mode>` | Payload Encryption. `0`: Off, `1`: On. |

### 2. Data Transmission to the Sigfox Network
| Command | Set Format | Description |
| :--- | :--- | :--- |
| `AT$SB` | `AT$SB=<Bit>,<Downlink>,<TxRepeat>` | Sends a dynamic status **bit** to the Sigfox network. |
| `AT$SF` | `AT$SF=<TextoASCII>,<Down>,<TxRpt>` | Sends an **ASCII Data frame** (E.g., `AT$SF=Hello`). |
| `AT$SH` | `AT$SH=<Len>,<HexPayload>,<Dwn>,<Rpt>` | Sends an **Hex frame** (Raw hex data). |
| `ATS300` | `ATS300` | Sends an *Out of Band* message exactly once. |

### 3. Regional Configuration (Macro-Channels) Sigfox MAC
| Command | Set Format | Description |
| :--- | :--- | :--- |
| `AT$RC` | `AT$RC=<Zone>` | Configures or reads the zone or Region. `Zone = [1 to 7]`. |
| `ATS302` | `ATS302=<Power>` | Setter Radio output power in dBm. `[0..20]`. |
| `ATS400` | `ATS400=<P1><P2><P3>,<P4>` | Standard specific variables (Mainly FCC). |
| `AT$RSSICAL` | `AT$RSSICAL=<Valor_dB>` | Get or set the RSSI calibration value in dB. |
| `ATE` | `ATE=<State>` | UART Echo state. `0` = Disable, `1` = Enable. |

### 4. Tests and Scan Modes (Sigfox)
*Note: Oriented to RF P1 / P2 certifications and testing.*
| Command | Format | Description |
| :--- | :--- | :--- |
| `AT$CW` | `AT$CW=<Freq>` | CW (Continuous Wave) test in given Hz or MHz frequency. |
| `AT$PN` | `AT$PN=<Freq>,<Bitrate>` | Starts PRBS9 test (BPSK pseudo-random test). |
| `AT$MN` | `AT$MN=<TimeOut>` | Starts *Monarch* scan for given time (Timeout sec). |
| `AT$TM` | `AT$TM=<rc>,<mode>` | Run Sigfox Test mode, `mode=[0..12]`. |
| `AT$RL` | `AT$RL` | Test: Starts listening loop (Local Loop cycle). |
| `AT$SL` | `AT$SL` | Test: Sends Local Loop Tx packet. |
| `AT$SP2P` | `AT$SP2P` | Test: Send data for P2P. |
| `AT$RP2P` | `AT$RP2P` | Test: Receive data for P2P. |
